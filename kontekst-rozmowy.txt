WIDOK SZCZEGÓŁOWY POKÉMONA (loadDetail → renderDetail)
================================================================

Sekcja 1: PORTRET + DANE PODSTAWOWE (lewa kolumna)
  - Artwork (official-artwork z PokeAPI)
  - Typy z kolorami (18 typów)
  - Wzrost, waga, habitat, capture rate, base exp
  - Zdolności (kolor: #55ffff zwykłe, #a855f7 ukryte)
  - Flavor text (angielski z PokeAPI)

Sekcja 2: STATYSTYKI BAZOWE (prawa kolumna)
  - Paski HP/ATK/DEF/SpATK/SpDEF/SPE z kolorami
  - Suma statystyk

Sekcja 3: IV (Individual Values) - skala 0-31
  - Funkcja buildIVSection(stats)
  - Automatyczna detekcja roli przez detectRole(statMap):
      isPhys  = ATK >= SpATK + 15
      isSpec  = SpATK >= ATK + 15
      isMixed = ani jedno
      isTank  = (DEF+SpDEF)/2 >= 90 && SPE < 80
      isFast  = SPE >= 80
      isTR    = SPE <= 50 (Trick Room kandydat)
  - CEL per stat (31 / 0 / >=20) z uzasadnieniem
  - ATK = 0 dla specjalnych atakujących (Foul Play / Confusion)
  - SpATK = 0 dla fizycznych atakujących
  - Pola input 0-31 z live paskami kolorowymi
  - Skala: 28-31=Doskonałe(zielony), 20-27=Dobre(żółty),
           10-19=Słabe(pomarańczowy), 0-9=Złe(czerwony)

Sekcja 4: COMPETITIVE BUILD (fioletowa sekcja)
  - Funkcja buildCompetitiveSection(stats, types, moves)
  
  4a. EV Spread (buildEVSection):
      - Rozkłady 252/252/4 lub 252/128/128 zależnie od roli
      - Przykłady:
          Spec Fast:   252 SpATK / 252 SPE / 4 HP
          Spec Tank:   252 HP / 252 SpDEF / 4 SpATK
          Phys Fast:   252 ATK / 252 SPE / 4 HP
          Trick Room:  252 ATK / 252 HP / 4 SPE
          Full Tank:   252 HP / 128 DEF / 128 SpDEF
      - Max 510 EV na Pokémona, max 252 na stat
  
  4b. Natura (buildNatureBox):
      - Fizyczny szybki → Jolly (+SPE / -SpATK)
      - Fizyczny wolny  → Adamant (+ATK / -SpATK)
      - Trick Room fiz  → Brave (+ATK / -SPE)
      - Spec szybki     → Timid (+SPE / -ATK)
      - Spec wolny      → Modest (+SpATK / -ATK)
      - Trick Room spec → Quiet (+SpATK / -SPE)
      - Spec Tank       → Calm (+SpDEF / -ATK)
      - Phys Tank       → Impish (+DEF / -SpATK)
  
  4c. Held Item (buildItemBox + HELD_ITEMS):
      - SPE>=100 + Phys  → Choice Band
      - SPE>=100 + Spec  → Choice Specs
      - Phys/Spec Fast   → Life Orb
      - Tank (Spec)      → Leftovers
      - Tank (Phys)      → Rocky Helmet
      - Mixed            → Life Orb
  
  4d. Best 4 Moveset (buildBest4):
      - Baza danych MOVE_DATA: ~120+ ataków z [power, type, category]
      - Algorytm scoringu:
          base_score = power
          × 1.4 jeśli kategoria pasuje do roli
          × 1.5 jeśli STAB (typ pokémona)
          + 25 pkt za priority moves
          setup moves: 80pkt ofensywne, 70pkt defensywne (tank), 15pkt utility
      - Wybór top 4 z diversyfikacją typów (max 2 same type)
      - Każda karta: nazwa, Fiz./Spec./Status, PWR, STAB/coverage, gwiazdki ⭐

Sekcja 5: EWOLUCJA (łańcuch klikalny)
  - renderEvoChain(chain) - rekurencja przez chain z PokeAPI
  - Trigger: Lv.X / item / trade / ♥ przyjaźń
  - Kliknięcie na formę przechodzi do jej detali

Sekcja 6: POLECANE DLA COBBLEMON (zielona sekcja)
  - Ewolucja przez przyjaźń: FRIENDSHIP_EVOS (16 Pokémonów)
      Komenda: /checkfriendship @s
      Próg: 160/255 friendship
  - Link Cable evolutions: LINK_CABLE_EVOS (21 Pokémonów)
      np. Kadabra→Alakazam (bez przedmiotu)
          Onix→Steelix (Metal Coat)
          Seadra→Kingdra (Dragon Scale)

Sekcja 7: PEŁNA LISTA ATAKÓW (zakładki)
  - "Przez poziom" - tabela z Lv, nazwa, typ, PWR
  - "TM/TR" - chipy z nazwami
  - "Egg Moves" - fioletowe chipy (jeśli są)

TECHNICZNE

- Global functions: updateIVBar(input), switchMovesTab(btn, panelId)
- Fonts: Google Fonts (VT323 + Press Start 2P)
- Nie używa żadnych zewnętrznych bibliotek JS
